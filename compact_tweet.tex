  \node[anchor=north west,fill=bluepale,inner sep=2.3mm,text width=141.4mm]
    at ([xshift=7mm,yshift=-19mm]current page.north west) {%
      {\color{blue}\bfseries\fontsize{8}{9}\selectfont CLAIM}\par
      \vspace{0.7mm}
      {\fontsize{10}{11.5}\selectfont
      For every $3$CNF formula $\phi$ with $n$ variables and $m$ clauses, one
      can construct a degree-$7$ polynomial map
      $K_\phi:\mathbb R^{3n+3m}\to\mathbb R^{3n+3m}$ such that
      \[
        K_\phi^{-1}(0)\ne\varnothing
        \iff \phi\in\mathrm{SAT},
        \qquad
        \det JK_\phi\equiv(-3456)^{n+m}.
      \]}
    };

  \node[anchor=north west,inner sep=0pt,text width=146mm]
    at ([xshift=7mm,yshift=-45.5mm]current page.north west) {%
      {\fontsize{7.6}{9}\selectfont
      \setlength{\abovedisplayskip}{1mm}
      \setlength{\belowdisplayskip}{1mm}
      \textbf{Proof idea.}
      Fix a polynomial map $H:\mathbb R^3\to\mathbb R^3$ with
      $\det JH\equiv-3456$, with exactly three preimages of zero,
      $H^{-1}(0)=\{r_0,r_+,r_-\}$, and with some omitted target
      $\Delta\notin\operatorname{im}(H)$.  The decoder
      $\beta(x,y,z)=x^2$ reads $r_0$ as $0$ and $r_\pm$ as $1$.

      \vspace{0.8mm}
      Given $\phi$, give each variable $X_i$ a block $w_i\in\mathbb R^3$ and
      include $H(w_i)$ in the output.  Therefore every zero forces
      $X_i=\beta(w_i)\in\{0,1\}$.  For each clause $C_j$, let $s_j$ be its
      usual polynomial Boolean value, so $s_j=1$ exactly when $C_j$ is true.

      \vspace{0.8mm}
      Give the clause a fresh block $v_j$ and output
      $T_{s_j}(v_j)=H(v_j)-(1-s_j)\Delta$.  Since $H$ reaches $0$ but never
      reaches $\Delta$,
      \[
        \exists v_j:\ T_{s_j}(v_j)=0
        \quad\Longleftrightarrow\quad s_j=1.
      \]

      Stack all variable and clause blocks to obtain $K_\phi$.  It has a zero
      exactly when every clause is true.  Moreover, each $T_{s_j}$ differs
      from $H$ only by an output translation, so $JK_\phi$ is block lower
      triangular with copies of $JH$ on the diagonal.  Hence
      $\det JK_\phi=(-3456)^{n+m}$, while $\deg K_\phi\le7$.}
    };
